# AttenTrack 📊

> Attendance Tracker & Simulator for college students. Fully client-side, powered by LocalStorage with optional Firebase Cloud Sync.

## Live Deployment
🚀 **[View Live on Vercel](https://attentrack-one.vercel.app/)**

---

## Features

- **Attendance Monitoring**: Track daily attendance percentage per subject relative to a target percentage.
- **Timetable Scheduling**: Configure weekly schedules with day and time slots.
- **What-If Simulator**: Predict how attending or skipping future classes will affect your attendance rate.
- **OCR Timetable Import**: Scan and upload your timetable from PDF or image (OCR powered by Tesseract.js).
- **Firebase Sync**: Optional Google Sign-In and Firestore synchronization across devices.

---

## Setup & Running Locally

1. Clone this repository.
2. Install the dev dependencies:
   ```bash
   npm install
   ```
3. Run a local server to open `index.html` (or use VS Code Live Server).
